# NEO Asteroid Analysis

Exploratory data analysis of NASA-certified Near-Earth Objects (NEOs).

## Dataset
90,836 asteroids with features including diameter, relative velocity, 
miss distance, and hazard classification.

## Contents
- `project2.ipynb` — Full analysis notebook
- `neo.csv` — Raw dataset sourced from Kaggle

## Questions Explored
- How frequently do asteroids of different sizes approach Earth?
- What proportion are classified as hazardous?
- Does size and velocity influence miss distance?
